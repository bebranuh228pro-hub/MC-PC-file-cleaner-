# MC-PC-file-cleaner-

🇷🇺 Описание на русском

🧹 Smart Cleaner Pro [Experimental]
Smart Cleaner Pro — это мощная утилита для автоматизированного поиска и безопасного удаления следов игрового софта на Minecraft. Программа разработана как экспериментальный образец для быстрого сканирования системы и поддержания чистоты на ПК.

✨ Основные возможности:

 🔍 Глубокий поиск: Сканирование системы по ключевым словам и расширениям файлов.

 🛡️ Шредер файлов: Безопасное удаление (перезапись нулями), исключающее возможность восстановления данных.
 
 💾 Сейф бэкапов: Автоматическое архивирование перед удалением важных данных.

 ⚡ Оптимизация: Очистка временных папок (Temp), кэша Discord, сброс DNS и очистка буфера обмена.

 🎮 Поддержка: Специализированная чистка логов и кэша Minecraft.

 📋 Инструкция: Встроенное меню для легкого управления всеми функциями.

⚠️ Важная информация:
 Экспериментальный образец: Программа предназначена только для тестирования.
 Язык: Интерфейс программы доступен только на русском языке.
 Права: Для корректной работы всех функций требуется запуск от имени администратора.

🇺🇸 English Description

🧹 Smart Cleaner Pro [Experimental]
Smart Cleaner Pro is a powerful utility designed for automated searching and secure deletion of traces left by Minecraft-related software. This tool is built as an experimental project to 
help users quickly scan their system and maintain a clean environment.

✨ Key Features:

🔍 Deep Scan: Search your entire system using custom keywords and specific file extensions.

 🛡️ File Shredder: Secure deletion (zero-fill overwrite), ensuring data cannot be recovered.

 💾 Backup Vault: Automatic file archiving before any deletion takes place.
 
 ⚡ Optimization: Clean temporary (Temp) folders, clear Discord cache, flush DNS, and clear the clipboard.

 🎮 Support: Specialized cleaning for Minecraft logs and game cache.

 📋 Built-in Guide: Interactive menu that makes managing the cleaning process effortless.

⚠️ Important Notes:
 Experimental Project: This software is intended for testing purposes only.
 Language: The user interface is available in Russian only.
 Privileges: Running the program as an Administrator is required for full functionality.
